# E_Commerce-sales
power bi project
This project is an end-to-end data analysis of an e-commerce dataset using SQL and Power BI. The goal was to understand how sales are performing, identify top products and categories, and get insights into customer behavior.
I worked on cleaning messy data (including missing values), writing SQL queries to extract insights, and finally building an interactive Power BI dashboard to present the results in a clear and visual way.
Used SQL to analyze sales, customers, and product performance
Created important metrics like total revenue, total orders, and average order value
Identified repeat customers and purchasing patterns
Built a Power BI dashboard with filters and interactive visuals
# Dashboard Highlights
KPI cards showing total revenue, orders, and average order value
Sales breakdown by category and city
Monthly sales trend
Payment method analysis
Top-selling products
Interactive filters (date, category, city)
